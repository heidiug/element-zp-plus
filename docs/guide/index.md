# 介绍

## Element ZP Plus

Element ZP Plus 是一个基于 Vue 3 的组件库，提供丰富的 UI 组件，帮助开发者快速构建高质量的 Web 应用。

## 特性

- **丰富的组件**：提供 10+ 个高质量组件，覆盖常用业务场景
- **基于 Vue 3**：使用 Vue 3 Composition API 构建，性能优异
- **TypeScript 支持**：完整的 TypeScript 类型定义，提供更好的开发体验
- **按需引入**：支持按需引入，减小打包体积
- **主题定制**：支持自定义主题，轻松适配不同设计风格
- **响应式设计**：组件支持响应式布局，适配各种屏幕尺寸

## 浏览器支持

现代浏览器和 IE11+（需要 polyfills）

## 版本

当前版本：v0.0.0
