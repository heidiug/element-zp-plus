# Alert API

## Attributes

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| --- | --- | --- | --- | --- |
| title | 标题 | `string` | — | — |
| type | 主题 | `string` | success / warning / info / error | info |
| description | 辅助性文字 | `string` | — | — |
| closable | 是否可关闭 | `boolean` | — | true |
| center | 文字是否居中 | `boolean` | — | false |
| show-icon | 是否显示图标 | `boolean` | — | false |
