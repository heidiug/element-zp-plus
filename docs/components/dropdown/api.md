# Dropdown API

## Dropdown Attributes

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| --- | --- | --- | --- | --- |
| trigger | 触发方式 | `string` | hover / click | hover |
| placement | 菜单弹出位置 | `string` | top / top-start / top-end / bottom / bottom-start / bottom-end | bottom |

## DropdownItem Attributes

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| --- | --- | --- | --- | --- |
| disabled | 是否禁用 | `boolean` | — | false |
| divided | 是否显示分隔符 | `boolean` | — | false |
